# Blueprint Operativo de AgenticCart API

## Reglas de Negocio Core
- **Productos:** Cada ítem debe contar obligatoriamente con un SKU único, un título optimizado y tags semánticos.
- **Seguridad** Ningun agente de IA puede modificar archivos del sistema sin una doble verificación del linter local.
- **Formato** Toda salida generada por inteligencia artificial destinada al usuario final debe ser estructurada y validada.
